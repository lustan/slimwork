# 执行实施计划

在已有书面计划时，按步骤落地并持续验证。

## 原则

- 计划优先，偏差可记录但不可无痕偏离
- 每个关键节点必须验证
- 每完成一段就更新状态，保证可追踪

## 建议流程

1. 复述目标与边界
2. 按顺序执行计划步骤
3. 每步完成后做最小验证
4. 关键阶段做代码质量审查
5. 汇总结果、风险与后续动作
