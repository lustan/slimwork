# Anthropic 最佳实践（中文版精简版）

本文档汇总在构建高质量技能与提示词时的关键实践。

## 1. 明确任务边界

- 先定义目标、非目标与完成标准
- 对术语给出一致定义
- 避免让模型推测隐含约束

## 2. 指令分层

- 全局原则：稳定不变的规则
- 任务流程：可执行步骤
- 结果格式：结构化输出约束

## 3. 降低歧义

- 使用可验证语言（可测、可复现）
- 给出正反例
- 对失败场景预设处理方式

## 4. 强化可执行性

- 每一步都应能落地
- 对关键节点给出检查清单
- 约束“先验证再结论”

## 5. 评估与迭代

- 用真实样例回放流程
- 记录失败模式并修正文档
- 持续减少无效上下文与重复指令
