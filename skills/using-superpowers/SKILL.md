# 使用 Superpowers 技能体系

本技能用于会话起始阶段，确保先识别并调用正确技能，再执行任务。

## 核心要求

- 先判断是否应触发技能
- 触发后先读取 `SKILL.md` 的必要内容
- 多技能场景只使用最小必要集合
- 输出过程与结果需可验证

## 目标

通过标准化技能调用，提升一致性、可复现性与交付质量。
